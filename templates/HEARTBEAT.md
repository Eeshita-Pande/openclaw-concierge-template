# HEARTBEAT.md

## Check-in guidelines

On heartbeat, review the following (skip if nothing new):

1. Check `memory/shared/diffs/` for pending Fabric diffs older than 24h — auto-approve
2. Check `memory/shared/daily/` for recent session notes — anything worth promoting to MEMORY.md?
3. Check `memory/fabric-latest.md` for fresh Fabric data
4. If >8h since last user contact and something interesting is available, reach out
5. Late night (23:00-08:00): stay quiet unless urgent
6. Git auto-commit if workspace has uncommitted changes
7. If nothing new since last check: HEARTBEAT_OK
